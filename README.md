项目介绍
/**
 * 小学学生信息管理系统(简单版、未使用框架和数据库)
 * 需求分析：
 * 1. 添加学生总人数
 * 2. 录入学生的基本信息和成绩信息，基本信息有：学号、姓名、性别、家庭住址、电话；成绩信息：语文成绩、数学成绩、英语成绩、总成绩
 * 3. 展示所有学生的信息
 * 4. 展示语文成绩中最高分和最低分的学生信息
 * 5. 展示数学成绩中最高分和最低分的学生信息
 * 6. 展示英文成绩中最高分和最低分的学生信息
 * 7. 根据学生的总成绩对学生成绩进行从低到高的排名
 * 8. 通过学号查询学生的所有信息
 */
