# -Java-
主要收录一些自己在学习Java的过程中自己所写的一些项目，这些项目有简单的一点的，都没有使用数据库和java框架，也有高级的项目包括使用一些java常见且重要的一些框架。


Project_1
项目介绍
小学学生信息管理系统(简单版、未使用框架和数据库)
需求分析：
1. 录入学生的基本信息和成绩信息
2. 展示所有学生的信息
3. 展示语文成绩中最高分和最低分的学生信息
4. 根据学生的总成绩对学生成绩进行从低到高的排名
5. 通过学号查询学生的所有信息
6. 退出系统
